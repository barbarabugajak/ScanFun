# ScanFun

> In this fast-paced game, you are the only cash register facing an endless stream of products.  
Match each item with the correctly colored scanner and brace yourself — every mistake might trigger bizarre side effects.  
What will it be this time... will the conveyor belt start glitching… or will the whole world turn black & white?
